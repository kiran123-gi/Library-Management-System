# 📚 Library Management System

A full-stack web application built using Django to manage library books efficiently.

## 🚀 Features
- Add new books to the library
- View all books with title, author, and quantity
- Store and manage data using SQLite database
- Simple and user-friendly interface using HTML & CSS
- Admin panel support for easy data management

## 🛠️ Tech Stack
- Python
- Django
- SQLite
- HTML5 & CSS3

## 📂 Project Modules
- Models: Book schema design
- Views: Business logic for adding and displaying books
- Templates: UI rendering using Django Template Engine

## ⚡ How to Run
1. Clone this repository
2. Navigate to project folder
3. Run: `python manage.py runserver`
4. Open browser: `http://127.0.0.1:8000/`

## 🎯 Learning Outcomes
- Understood Django MVT architecture
- Learned database integration using ORM
- Built dynamic web pages using templates
- Gained hands-on experience in full-stack development
